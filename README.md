# 競艇 買い目予想(5場)

対象5場(江戸川・平和島・常滑・尼崎・若松)の買い目を、機械学習(LightGBM)で予測して
毎日レポートを生成するシステム。

## スマホでの見方

GitHub Pages で公開された予想ページを開くだけ:

`https://romancedawn59.github.io/<リポジトリ名>/`

このURLをホーム画面に追加すればアプリのように使える。毎日朝6:30ごろ自動更新。

## 手動で実行(スマホ/PC)

GitHubの **Actions** タブ →「予想レポート更新」→「Run workflow」で、
好きなタイミングに収集+予想を走らせられる(PCの電源は不要)。

## 仕組み(クラウド版)

- **予想レポート更新**(毎日 / 手動): データ収集 → 予測 → Pages公開
- **モデル再学習**(毎月1日 / 手動): 最新1年で再学習 → バックテスト検証

DB(SQLite・約137MB)はリポジトリに含めず、GitHub Actionsのキャッシュで実行間を持ち回る
(キャッシュが無い初回は約1年分を自動で作り直す)。

## 買い目の考え方

ウォークフォワード検証で期待値プラスが確認できた「荒れ注意」レース
(予測1位艇の勝率35%未満)のみを「勝負」とし、1レース1,000円で
3連複軸1流し3点(600円)+3連単の中位艇頭2点(400円)を推奨する。
堅め・標準レースは期待値マイナスのため見送り。

詳細な仕組みは `flow.html`、使い方は `manual.html` を参照。

## ローカルでの実行

```
cd src
python collect.py          # データ収集
python predict.py today    # 今日の予想レポート生成(reports/へ)
python train_model.py      # モデル再学習
python backtest.py         # 検証
python -m unittest discover ../tests
```

※ データ源はいずれも非公式/公開情報(BoatraceOpenAPI, Open-Meteo, boatrace.jp)。
予測は参考情報であり的中を保証しない。舟券購入は自己責任で。
